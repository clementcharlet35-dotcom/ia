# Pronostics MVC

## Installation

1. Copier `.env.example` vers `.env`
2. Configurer la base de données
3. Importer `database/schema.sql`
4. Installer les dépendances :

```bash
composer install
```

5. Lancer le serveur :

```bash
php -S localhost:8000
```
